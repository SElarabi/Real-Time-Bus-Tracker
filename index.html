# BUS NEXT STOP


Description

This webage is run by a Javascript code and displays the real-time location of buses as icons on a map using Mapbox API. It retrieves data from an MBTA API and updates the markers on the map every 15 seconds to show the current locations of the buses.
The code uses the fetch API for data retrieval and the Mapbox API for map and marker manipulation.
